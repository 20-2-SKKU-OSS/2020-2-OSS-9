Contributing to an Open Source Project [free-python-games]

free-python-games

(project link free-python-games github link)
Free Python Games는 교육과 재미를 목적으로 하는 무료 Python 게임의 Apache2 라이선스 모음입니다. 이 게임들은 간단한 python 코드로 쓰여지고 실험과 변화를 위해 고안되었습니다. 몇몇 클래식 아케이드 게임의 단순화된 버전이 포함되어 있습니다. 2012년부터 방과후 학교 프로그램으로 시작되어 free python games은 도심 속 청소년들에게 프로그래밍을 가르쳤습니다. 배우는 만큼 재미있게 노는 것이 목표였고, 게임은 지속적으로 향상되어 교실에서부터 여름 데이 캠프에 이르는 다양한 환경에서 사용되었습니다. 각 게임은 다른 게임과는 완전히 독립적이며 학생들과 함께 연습할 수 있는 목록과 함께 코멘트를 포함합니다. 창의성과 유연성이 중요하며 새로운 특징이나 행동을 구현하는 옳고 그른 방법은 없습니다! .

link 따라서 9조는 학생들이 다양한 게임으로 프로그래밍 교육을 배우며 즐거움을 느낄 수 있도록 현재 게임들 중에 없는 새로운 파이썬 게임을 함께 개발하고, 그에 대한 게임 방법과 교육 문서등을 작성하여 기여하고자 합니다.
