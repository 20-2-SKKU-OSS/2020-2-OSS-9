OSS-9의 프로젝트는 `free python games`에 기여하는 것입니다. `Free Python Games` 는 교육과 놀이를 목적으로 하는 파이썬 게임 모음입니다. 이 게임은 간단한 파이썬 코드로 제작된 미니 게임, 고전 게임들로 구성되어 있습니다. Team OSS-9은 이 게임에 새로운 게임인 `스도쿠`를 추가하여 Free Python Games의 개발자에게 pull request하고 Free Python Games에 기여하는 것을 목표로 합니다.
